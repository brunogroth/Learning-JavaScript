Formulário responsivo com Flexbox

O LayoutFlexible Layout Box Model, mais conhecido como Flexbox, é uma ótima ferramenta para criar áreas responsivas ou elementos de interface do usuário. Embora não seja adequado para layouts de página complexos, o Flexbox se destaca no controle de elementos em um único eixo ou na organização de elementos em pequenas áreas. Isso torna o Flexbox uma ferramenta fantástica para a criação de elementos de página responsivos que normalmente são difíceis de manusear, como formulários. Freqüentemente, o Flexbox pode criar componentes responsivos com uma quantidade mínima de código.
